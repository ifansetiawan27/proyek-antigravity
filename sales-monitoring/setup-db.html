Diagnostik
1 Cek Status Tabel Supabase

Klik tombol di bawah untuk memeriksa tabel mana yang sudah ada di proyek Supabase Anda.

2 Buat Tabel Otomatis via Management API
Untuk membuat tabel secara otomatis, dibutuhkan Supabase Personal Access Token (bukan anon key).

Cara mendapatkannya:
  1. Login ke supabase.com/dashboard
  2. Klik foto profil (pojok kiri bawah) → Access Tokens
  3. Klik "Generate new token" → beri nama → Generate
  4. Copy token yang muncul dan paste di bawah
// Log akan muncul di sini setelah proses dimulai...
3 Sinkronisasi Data Awal ke Supabase

Tabel berhasil dibuat! Sekarang kembali ke aplikasi utama dan klik tombol Sync di topbar untuk mengirim data lokal ke Supabase.
